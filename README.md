
# Pigs Plus

Before horses, there were pigs. Pigs Plus updates the ability to ride pigs by adding a Golden Carrot on a Stick for faster pig transportation, the ability to swim while riding a pig, and more. 


## Features

- Golden Carrot on a Stick - A faster alternative to the carrot on a stick, also with much higher durability.
- Swim while riding a pig by using the Golden Carrot on a Stick.
- Pigs can now breed & heal using golden carrots.
- [Coming Soon] Pig Armor
